# terlogin

termux login v1.2

cara install :

* pkg update && pkg upgrade
* pkg install git
* git clone https://github.com/JLynx69/terlogin
* cd terlogin
* sh install.sh
* edit username dan passwordnya
* mulai ulang termux
* Enjoy :)

Uninstall :

* cd terlogin
* chmod +x uninstall
* ./uninstall
