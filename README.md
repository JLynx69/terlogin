# terlogin

termux login v1.2

cara install :

* pkg update && pkg upgrade
* pkg install git
* git clone https://github.com/JLynx69/terlogin
* cd terlogin
* chmod +x *
* ./install
* edit username dan passwordnya
* mulai ulang termux
* Enjoy :)

Uninstall :

* cd terlogin
* ./uninstall
